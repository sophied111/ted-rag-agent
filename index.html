API Endpoints

POST/api/prompt

Query the RAG system with natural language questions about TED Talks.

Request Body:

{
  "question": "Your natural language question here"
}

Example:

curl -X POST https://your-app.vercel.app/api/prompt \
  -H "Content-Type: application/json" \
  -d '{"question": "Find a TED talk about brain and consciousness"}'

Response:

{
  "response": "AI-generated answer based on TED dataset",
  "context": [
    {
      "talk_id": "2817",
      "title": "Talk title",
      "chunk": "Retrieved text chunk",
      "score": 0.8543
    }
  ],
  "Augmented_prompt": {
    "System": "System prompt used",
    "User": "Full user prompt with context"
  }
}

GET/api/stats

Get current RAG configuration parameters.

Example:

curl https://your-app.vercel.app/api/stats

Response:

{
  "chunk_size": 1024,
  "overlap_ratio": 0.2,
  "top_k": 10
}
